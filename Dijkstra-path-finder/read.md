# Dijkstra Path Finder

This project visualizes the Dijkstra Algorithm using a grid-based system.

## Features
- Click to add obstacles
- Select source and destination
- Finds shortest path
- Animated visualization

## Technologies Used
- HTML, CSS, JavaScript
- Dijkstra Algorithm (DSA)

## How to Run
Open index.html in browser.

## Algorithm Used
Dijkstra's Algorithm using priority queue for shortest path calculation.
